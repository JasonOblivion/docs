## Installation
**Use git and pip to quickly install the LLM-VM**

Navigate to your desired directory and run the following

```bash installing anarchy
> git clone https://github.com/anarchy-ai/LLM-VM.git
> cd LLM-VM
> pip3 install .
```
This will install both the library and test-sever
